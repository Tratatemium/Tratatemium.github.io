# Personal Portfolio Website

A modern, responsive personal portfolio website showcasing my work, skills, and hobbies as a front-end developer.

## Features

- **Multi-page Navigation**: Home, About, Projects, and Hobbies sections
- **Dark/Light Mode**: Theme toggle for user preference
- **Image Lightbox**: Gallery viewer for project and hobby images
- **Smooth Animations**: Scroll-triggered animations with intersection observer
- **Responsive Design**: Mobile-friendly layout with CSS modules

## Tech Stack

- React 19
- React Router DOM
- CSS Modules
- TypeScript (select components)

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm start
```
Opens at [http://localhost:3000](http://localhost:3000)

### Build for Production
```bash
npm run build
```

## Project Structure

- `/src/pages` - Main page components (Home, About, Projects, Hobbies)
- `/src/components` - Reusable UI components (Navbar, Footer, Lightbox, etc.)
- `/src/hooks` - Custom React hooks
- `/src/styles` - Global styles and CSS variables
- `/src/assets` - Images and static resources
